# Content Enhancement Runbooks

Frameworks for improving existing content.

## Available Runbooks

| Runbook | Purpose |
|---------|---------|
| [blog-revitalization-framework.md](blog-revitalization-framework.md) | Comprehensive framework for updating historical blog posts |
| [blog-revitalization-prompt.md](blog-revitalization-prompt.md) | Quick prompt to use with the framework |

## Usage

The blog revitalization framework helps update old posts while maintaining temporal integrity - the enhanced version should read as if written on the original publication date, just with perfect clarity and execution.
