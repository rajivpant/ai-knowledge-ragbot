# System Configuration Runbooks

Prompts and configurations for controlling AI behavior.

## Available Runbooks

| Runbook | Purpose |
|---------|---------|
| [anti-watermarking.md](anti-watermarking.md) | Request clean text output without hidden markers |
| [code-generation.md](code-generation.md) | Structured approach to code tasks with multiple approaches |
| [llm-project-setup.md](llm-project-setup.md) | Configure Claude Projects, ChatGPT GPTs, and Gemini Gems |
| [response-synthesis.md](response-synthesis.md) | Combine multiple LLM responses into unified output |

## Usage

These runbooks are designed to be:
- Included in custom instructions for persistent behavior
- Used as prompts for specific tasks
- Adapted for your specific needs
