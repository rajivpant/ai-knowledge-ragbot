# Engineering Runbooks

Runbooks for software engineering practices and code quality.

## Available Runbooks

| Runbook | Purpose |
|---------|---------|
| [codebase-review.md](codebase-review.md) | Enterprise-grade codebase review checklist with tiered approach |

## Usage

The codebase review runbook is designed for use with AI coding assistants (Claude Code, Cursor, etc.) to conduct comprehensive code audits. It includes:

- **Tiered system**: Essential, Standard, Enterprise, Mission-Critical
- **Project complexity assessment**: Determine which tier applies
- **Minimum viable review**: 15-minute quick health check
- **Industry addenda**: Financial services, healthcare, e-commerce, government
- **Open source addendum**: Additional checks for OSS projects
